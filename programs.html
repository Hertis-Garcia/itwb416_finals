CICS — Academic Programs (BSIT & BSCS)

BS Information Technology (BSIT)

Program Description

The BSIT program trains students in practical information technology skills: systems administration, network management, software tools, and applied computing. Emphasis is placed on hands-on projects, industry-standard tools, and preparing graduates for implementation and support roles in organizations.

Curriculum Overview
Core areas:
  • Programming & Web Development
  • Database Systems & Administration
  • Networking & System Administration
  • Cloud Fundamentals & DevOps basics
  • Capstone: Implementing IT solutions for real-world clients

Sample sequencing: Foundation courses (years 1–2) → specialization & labs (year 3) → capstone & internships (year 4).

Career Paths
  • IT Support / Helpdesk Engineer
  • Network Administrator
  • Database Administrator (entry)
  • Systems Analyst / IT Operations
  • Cloud Support Engineer / DevOps Associate
  • Web & Application Support

BS Computer Science (BSCS)

Program Description

BSCS focuses on computational theory, algorithms, and software engineering. Students develop strong foundations in programming, data structures, algorithms, and software design along with opportunities for research, advanced topics (AI, data science), and capstone projects.

Curriculum Overview
Core areas:
  • Programming, Data Structures & Algorithms
  • Software Engineering & System Design
  • Theory of Computation & Discrete Math
  • Optional tracks: Artificial Intelligence, Data Science, Cybersecurity
  • Capstone: Software engineering project with industry or research focus

Sample sequencing: Foundations (years 1–2) → advanced algorithms & electives (year 3) → capstone & internship (year 4).

Career Paths
  • Software Engineer / Developer
  • Machine Learning Engineer / Data Scientist (with elective focus)
  • Systems Architect / Backend Engineer
  • Quality Assurance / Test Engineer
  • Security Analyst (with electives)
  • Research & Development roles or graduate studies
